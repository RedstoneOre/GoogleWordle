# Google Wordle

A fun interactive word-guessing game inspired by Wordle, powered by Google search results!

## Overview

Google Wordle is a web-based game that transforms Google search results into a Wordle-style guessing challenge. The game extracts words from various search result types and challenges players to guess them within a limited number of attempts.

## How to Play

### Starting a New Game

1. **Get an API Key**:
   - Register/login to [SerpAPI](https://serpapi.com)
   - Copy your API key

2. **Start the Game**:
   - Paste your API key into the "API key" input field
   - Enter a search prompt (or click the random button for suggestions)
   - Click the "Search!" button

3. **Make Guesses**:
   - Type your guess into the input field
   - Click the checkmark button to submit
   - See results in the guess history

### Loading Existing Games

- Copy previously saved game data
- Paste it into the data box at the top
- Continue from where you left off

## Game Statistics

Track your performance with:
- **Guesses**: Total number of guesses made
- **Guessed Words**: Number of successfully guessed words
- **Achievement Titles**: Titles earned during gameplay
- **Tips**: Hints or tips available

## Features

- Copy game data to clipboard for sharing
- Base64 encoding for game data persistence
- Responsive design for various screen sizes
- Support for multiple search result types
- Real-time game state management
